# 【职教通】Vocational Bridge V1.0 - 项目发布说明 (Release Notes)

## 1. 项目愿景 (Vision)
“职教通”是一款专为广东省中职生量身定制的 AI 学习与职业成长助手。我们致力于通过 AI 技术，消除教育资源差，为学生提供“提分外挂”，同时作为“职场护身符”保障其实习安全。

## 2. 核心功能清单 (Features)

### 🚀 2.1 AI 启发式讲题 (Socratic Tutor)
- **非填鸭式教育**：AI 导师严禁直接给出题目答案，而是通过“大白话”引导学生思考。
- **降维打击策略**：针对基础薄弱的学生，提供 3 级逻辑拆解。
- **Token 降本增效**：集成本地缓存与 Prompt 压缩技术，API 成本降低 30% 以上。

### 🛡️ 2.2 实习合同 AI 审计 (Contract Shield)
- **风险秒识别**：拍照上传实习合同，AI 自动扫描“非法押金”、“扣押证件”、“违规加班”等法律陷阱。
- **本地预过滤**：通过本地正则引擎实现 0 成本快速扫描，高危风险才调用 AI 深度审计。

### 📊 2.3 数据验证简历背书 (AI Resume Engine)
- **数据翻译官**：将学生的刷题正确率、复盘频率、学习时长自动翻译为 HR 看得懂的“职业素养评价”。
- **能力雷达图**：基于真实学习数据生成“逻辑、稳定性、抗压复盘力”五维能力模型。

### 🔍 2.4 风险规避型智能推荐 (Smart Internship Match)
- **全网信用库**：集成首批 50 家深圳本地企业信用分库。
- **主动避雷**：搜索到红/橙灯高风险企业时，AI 立即弹出预警并推荐同行业的绿灯优质企业。

### 💰 2.5 商业化闭环 (Commercial Loop)
- **分级会员体系**：支持基础版、提分版（讲题不限）、就业版（VIP 审计与简历优化）。
- **隐私优先注册**：用户名必填，真实姓名选填，仅在投递环节引导补全实名。

## 3. 技术栈 (Technology Stack)
- **前端 (Mobile)**: Flutter (Dart) - 极致的跨平台体验。
- **后端 (Server)**: NestJS (TypeScript/Node.js) - 企业级高并发架构。
- **数据库 (DB)**: PostgreSQL - 结构化存储学生能力数据。
- **AI 核心**: OpenAI GPT-4o / GPT-4o Vision - 多模态理解与推理。
- **监控**: 自研 `TokenTracker` 模块 - 实时 API 成本审计。

## 4. 安装与使用说明 (Installation)

### 4.1 环境要求
- Node.js >= 18.0
- Flutter SDK >= 3.0
- PostgreSQL 数据库

### 4.2 快速启动 (后端)
1. 进入 `src/backend` 目录。
2. `npm install` 安装依赖。
3. 在 `.env` 中配置 `OPENAI_API_KEY`。
4. `npm start` 启动服务。

### 4.3 快速启动 (前端)
1. 进入 `src/frontend` 目录。
2. `flutter pub get` 获取包。
3. `flutter run` 在模拟器中查看效果。

## 5. 联调测试结论 (RC1.0 Test Result)
- **全链路测试**：✅ 通过 (注册 -> 讲题 -> 数据累积 -> 简历生成 -> 避雷)。
- **性能指标**：AI 平均首字响应 < 1.5s，端到端完整报告生成 < 4.0s。
- **安全审计**：敏感信息本地加密处理，Token 消耗符合预期范围。
