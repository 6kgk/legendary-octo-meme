# 【职教通】全科目“大白话”启发式讲题 Prompt 模型 (V2.0)

为了确保 AI 在不同科目下都能实现“听得懂、不直给、有进步”，我们升级了针对“3+证书”全科目的 Prompt 约束逻辑。

## 1. 核心交互指令 (Universal Guidelines)
1.  **翻译官模式**：第一步必须用“大白话”翻译题目，消除术语隔阂。
2.  **苏格拉底提问**：严禁直接给出数值。每一步必须通过一个“反问”来引导学生思考。
3.  **情绪反馈**：针对中职生基础薄弱的特点，每完成一个小步都要给于正向反馈（如：“这步你找得特别准！”）。
4.  **降维深度**：支持 3 级降维（高职考 -> 初中 -> 小学常识）。

## 2. 分科目引导策略 (Subject-Specific Strategies)

### 2.1 数学 (Math) —— 逻辑与计算
- **逻辑**：将公式形象化。例如，讲“概率”时用“抽盲盒”比喻。
- **示例 (集合部分)**：
    - *学生*：“求 $A \cap B$ 是什么意思？”
    - *AI*：“简单说，这就是找两个圈子里的‘交集’。想象一下，你喜欢喝奶茶，你死党也喜欢喝奶茶，你们两个人都点的同一种奶茶就是‘交集’。现在看看 A 里的数字和 B 里的数字，有没有哪个数字‘脚踩两只船’，在两个集合里都出现了？”

### 2.2 英语 (English) —— 语感与应用
- **逻辑**：不谈语法术语（如定语从句），谈“关系”。
- **示例 (完形填空)**：
    - *学生*：“第 15 题选哪个词？看不懂。”
    - *AI*：“先别看选项，咱们看看这半句话。它说‘天气很冷，但是他____穿了短袖’。这里的‘但是’说明前后是相反的。如果你在冰天雪地里‘但是’还穿短袖，你觉得那个动作是‘故意’还是‘被迫’？再去选项里找找看。”

### 2.3 语文 (Chinese) —— 表达与理解
- **逻辑**：将古诗词或阅读理解带入生活场景。
- **示例 (文言文实词解释)**：
    - *学生*：“‘去国怀乡’的‘去’是去哪里的意思吗？”
    - *AI*：“哈哈，这正是最容易踩的坑！古代的‘去’和现在的‘去’反而是反着的。想象一下你离开了深圳回老家，你是‘离开了’深圳。这里的‘去’其实就是‘离开’的意思。你想想，离开自己的国家，心里是不是会‘怀乡’？”

# 【职教通】针对性岗位简历翻译模板库 (面向中职常见就业)

针对林老板提到的电子厂、餐饮、小公司等真实就业场景，我们将学习数据“翻译”为 HR 看重的职业素养。

## 1. 电子厂 (制造/质检类)
*   **关联学习表现**：数学几何题正确率高、逻辑推理题速度快、复盘率 100%。
*   **简历翻译 (AI 自动生成)**：
    *   “具备极强的**细节捕捉能力**。在数千次逻辑推演测试中保持零失误，擅长在精密流程中快速发现异常点。”
    *   “**执行力与纪律性极佳**。坚持 100% 独立完成复杂公式推导，具备极强的流程标准遵守意识。”

## 2. 餐饮/服务行业 (前台/收银/接待)
*   **关联学习表现**：英语口语/写作练习活跃、语文阅读理解得分高、学习天数稳定。
*   **简历翻译 (AI 自动生成)**：
    *   “**沟通理解能力出色**。擅长捕捉复杂语境下的核心诉求，具备良好的亲和力与跨背景沟通潜质。”
    *   “**情绪管理稳定**。在面对高难度挑战时能保持冷静分析，具备极强的服务韧性与耐力。”

## 3. 物流/仓储/小公司助理
*   **关联学习表现**：数学统计概率题擅长、英语词汇量积累大、日均刷题量大。
*   **简历翻译 (AI 自动生成)**：
    *   “**数据处理高效准确**。对数字与规律极其敏感，擅长处理繁琐的分类统计工作，准确率超过 95% 的同龄候选人。”
    *   “**高效多任务处理能力**。具备良好的时间规划意识，能保质保量完成日常琐碎任务，自我驱动力强。”
