# 职教通 (Vocational Bridge) - 教师端使用手册 (V1.0)

## 1. 简介
职教通教师端是为中职班主任量身打造的“智慧管理中枢”。它集成了提分预警、实习避雷、家校评价、及自动化报表功能，旨在将老师从繁琐的数据统计中解放出来，专注于学生成长干预。

## 2. 核心功能场景

### 场景 A：周五下午“一键交账”
*   **痛点**：每周需要向学校教导处提交学生学情报表。
*   **操作**：登录 `http://localhost:3000/teacher/C001`，点击右上角 **“一键导出交账报表 (CSV)”**。
*   **产出**：自动生成的 Excel 表格，包含姓名、当前预估分、各科掌握度及专业课进度。

### 场景 B：识别“掉队”学生并干预 (Task #60)
*   **操作**：查看 **“学情分析卡片”**。
*   **预警逻辑**：预估分低于 200 分的学生将自动进入“挂科预警”列表。
*   **干预**：点击 **“一键分发弱点强化包”**，系统会自动根据全班薄弱点（如数学）推送专项练习。

### 场景 C：实习高危单位紧急拦截 (Task #64)
*   **操作**：关注 **“风险预警大厅”** 中的红色高危条目。
*   **逻辑**：当学生咨询或签约“红灯”单位（存在扣证、押金风险）时，系统自动触发拦截。
*   **话术模板**：点击 **“查看话术模板”**，系统会提供专业的劝导话术，帮助老师与学生进行风险谈话。

### 场景 D：家校协同谈话 (Task #66)
*   **操作**：在学生明细表中查看 **“家校协同动态”**。
*   **逻辑**：家长端发放的“专注/进步/韧性”勋章及留言会自动同步。
*   **应用**：谈话时可以引用：“XX同学，我看到你妈妈上周给你发了‘专注勋章’，说你在家复习很认真，老师也觉得你最近状态不错...”

## 3. 常见问题 (FAQ)
*   **Q: 预估分是如何计算的？**
    *   A: 基于 AI 辅导过程中的正确率、思考深度及最近模拟考成绩综合得出。
*   **Q: 谈话记录归档后可以修改吗？**
    *   A: 目前设计为不可修改，直接进入“成长档案”，作为学生职业素养评价的重要依据。
