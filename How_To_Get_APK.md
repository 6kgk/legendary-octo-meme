# 如何获取您的“职教通”手机安装包 (APK)

林老板，由于当前开发环境不具备手机打包工具（Android SDK），我们为您配置了最专业的**“云端自动打包”**方案。您无需配置任何复杂的技术环境，只需通过 GitHub 即可一键获得安装包。

## 第一步：创建 GitHub 仓库
1. 登录您的 GitHub 账号（如果没有，请在 [github.com](https://github.com) 注册）。
2. 创建一个新的私有或公开仓库，命名为 `vocational-bridge`。
3. 将项目代码（`D:\CampusAppProject\src\frontend` 目录下的所有内容）推送到该仓库。

## 第二步：触发自动构建
1. 代码推送成功后，点击仓库顶部的 **Actions** 标签。
2. 您会看到一个名为 **"Build APK"** 的工作流。
3. 每当您修改代码并推送，或者点击 "Run workflow" 按钮时，GitHub 的云端服务器会自动开始为您打包。

## 第三步：下载安装包
1. 等待大约 3-5 分钟，构建完成后会显示绿色的对勾。
2. 点击该次构建记录，在页面下方的 **Artifacts** 区域，您会看到一个名为 `app-release` 的文件。
3. 下载并解压，即可得到 `app-release.apk`。
4. 将此文件发送到手机上，安装即可开始内测！
