# 【职教通】V2.0 商业级重构技术白皮书

## 1. 核心目标
对标行业头部应用（作业帮、小猿搜题），实现从“UI 驱动”向“内核驱动”的跨代升级。重点突破：认知溯源引擎、三段式交互、离线高性能索引。

## 2. 底层架构 (The Foundation)
采用 **Clean Architecture (简洁架构)** 分层：
- **Data Layer (数据层)**：引入 Repository 模式，统一管理 SQLite 本地库与远程 API，实现多级缓存。
- **Domain Layer (领域层)**：纯逻辑层，处理艾宾浩斯记忆算法、职业生命力评估算法。
- **Presentation Layer (表现层)**：基于 BLoC 或 StateNotifier 的响应式 UI，确保状态全屏同步。

## 3. AI 辅导内核 (The Socratic Brain)
### 3.1 三段式交互逻辑
1. **意图锚定 (Intent Anchoring)**：拍照后先询问题目要求，确认学生对“题眼”的识别。
2. **认知溯源 (Cognitive Trace)**：自动比对知识关联树，定位底层断层（如：二次函数 -> 一元二次方程 -> 基本代数）。
3. **分步降维 (Step-by-step Downgrade)**：采用“大白话”模型，分 3-5 步进行启发式引导。

## 4. 性能指标 (SLA)
- **拍搜响应**：本地特征码索引 < 100ms。
- **讲题首字**：流式 SSE 输出 < 300ms。
- **数据同步**：全局状态刷新 < 16ms (60fps)。

## 5. 职业化演进 (Career Evolution)
- **行业大师 AI**：集成 20+ 行业专家人格，提供课本外的实战潜规则。
- **名企绿通**：基于学情数据的“AI 验证自荐信”直接挂钩企业 HR 接口（模拟）。
