# 广东省“3+证书”高职高考标准教学路径库 (AI 核心逻辑版)

## 一、 研发背景与指导思想
本路径库基于广东省教育考试院 2024-2026 年《“3+证书”考试说明》进行结构化重组。其核心设计目标是为 AI 导师提供一套**“由点到面、阶梯上升”**的标准化辅导逻辑，确保 AI 在与学生交互时，能够精准定位知识断层并提供针对性引导。

---

## 二、 学科知识图谱与教学路径

### 1. 数学 (Mathematics): 逻辑与应用路径
**核心理念**：以“基础运算”为根基，以“函数思维”为核心，以“空间与概率”为拓展。

| 模块 | 关键知识点 (Knowledge Points) | AI 引导阶梯 (Teaching Steps) | 职业映射 (Career Mapping) |
| :--- | :--- | :--- | :--- |
| **基础逻辑** | 集合运算、充要条件、不等式 | 1. 直观比喻 (如: 分水果) -> 2. 符号语言 -> 3. 混合运算 | 基础逻辑判断 |
| **函数基石** | 二次函数、指数/对数、函数图像 | 1. 变量对应 -> 2. 图像特征识别 -> 3. 参数调节(动点) | 参数调试/质检 |
| **三角几何** | 三角比、正余弦定理、空间几何体 | 1. 单位圆理解 -> 2. 公式推导 -> 3. 实际测绘应用 | 工程测绘/精密制造 |
| **数列与概率** | 等差/等比数列、古典概率、统计 | 1. 规律发现 -> 2. 通项归纳 -> 3. 数据分布分析 | 跨境电商/仓储管理 |

### 2. 英语 (English): 语境与功能路径
**核心理念**：淡化语法术语，强化“语义块”理解与“应用文”写作。

| 模块 | 核心能力 (Core Competence) | 教学引导重点 | 职业映射 |
| :--- | :--- | :--- | :--- |
| **词汇认知** | 常用词汇 2500+ (大纲要求) | 从“单词含义”到“语境搭配”的联想记忆 | 文档阅读 |
| **语法结构** | 五大基本句型、时态转换、被动语态 | 关注“动作发生的时间”与“谁被做了什么”，避免术语堆砌 | 合规性理解 |
| **应用写作** | 书信、通知、感谢信、邮件 | 模版引导 -> 关键信息填空 -> 句式润色 | 跨境客服/行政助理 |

### 3. 语文 (Chinese): 素养与表达路径
**核心理念**：以“现代文阅读”提升理解力，以“应用文/议论文”训练表达逻辑。

| 模块 | 核心考点 | AI 启发逻辑 | 职业映射 |
| :--- | :--- | :--- | :--- |
| **语言应用** | 字音字形、成语辨析、病句修改 | 错误点识别 -> 原则解析 -> 修正训练 | 文案纠错/沟通能力 |
| **阅读理解** | 文言文实词/虚词、现代文逻辑 | 关键信息定位 -> 段落大意概括 -> 深度主旨理解 | 任务理解/信息提取 |
| **写作训练** | 议论文、记叙文、常用应用文 | 审题引导 -> 提纲构建 -> 论据充实 | 方案策划/工作汇报 |

## 三、 AI 引导规则 (SOP)
1.  **分级引导**：优先确认前置知识点（如求函数顶点前，先确认系数 a, b, c 的识别）。
2.  **降维策略**：若学生在当前难度停留超过 2 分钟，自动切换至“降维讲解”（如将二次函数类比为“投篮轨迹”）。
3.  **反馈闭环**：每一步正确操作后，记录 1 个“职业技能点”，并在简历生成器中实时累加。
