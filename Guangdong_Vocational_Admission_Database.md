# 2026 广东高职院校录取数据权威库 (教研参考版)

## 一、 数据概况
本数据库整合了 2024-2025 年广东省“3+证书”高职高考的真实录取数据。涵盖了本科批次、专科批次（中职生类、美术类、体育类）的核心指标，旨在为 AI 提分大脑提供精准的“分差预测”支持。

**核心控制线 (2025 估算)**：
*   **本科批次**：260 分
*   **专科批次**：120 分

## 四、 提分预测调用逻辑 (AI Logic)
1.  **分差计算**：`分差 = 目标院校最低分 - 当前预测分`。
2.  **概率评估**：
    *   `分差 <= -10`: **稳操胜券** (建议冲刺更高目标)。
    *   `-10 < 分差 <= 5`: **很有希望** (需稳住基础)。
    *   `5 < 分差 <= 20`: **冲刺可及** (需专项突破)。
    *   `分差 > 20`: **仍需努力** (需全方位提分)。
